design typography

This page showcases the typography system used on this site. All values are configurable via data/typography.yaml.

Typography Scale

The type scale uses a 1.25 (Major Third) ratio, where each step is 1.25× the previous:

ScaleSizeUse
xs0.64remCaptions, labels
sm0.8remSmall text, metadata
base1remBody text
md1.25remLead text, h4
lg1.563remh3
xl1.953remh2
xxl2.441remh1

Body Text

This is regular body text set in Inter at 18px (1rem) with a line height of 1.618 (the golden ratio). The maximum line width is constrained to 36rem (~65 characters) for optimal readability.

Italic text uses EB Garamond for elegant emphasis. Bold text uses a weight of 600.

Spacing

Spacing follows the golden ratio (φ = 1.618):

  • xs: 0.382rem (6.9px)
  • sm: 0.618rem (11px)
  • base: 1rem (18px)
  • md: 1.618rem (29px) ← golden ratio
  • lg: 2.618rem (47px) ← φ²
  • xl: 4.236rem (76px) ← φ³

Code

Inline code uses JetBrains Mono.

// Code block example
function goldenRatio(n) {
  const phi = (1 + Math.sqrt(5)) / 2; // 1.618...
  return n * phi;
}

Blockquote

The golden ratio appears throughout nature, art, and architecture. Using it for typography creates naturally pleasing proportions.
